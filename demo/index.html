Created to offer an easily modifiable grid base system zGrid is a fluid grid system. Nested grids. Modifiable. Set your margin spacing, columns, breakpoints, whether you want it responsive. Lightweight. This is a pure responsive grid layout. Not twitter bootstrap.

Download Github

Features

Responsive (or not)

Built for responsive designs, zGrid allows developers to build websites on a foundation suited for mobile and tablet browsers. But if you also want a grid system without a responsive design you can have that too.

Lightweight

No fonts. Styled buttons and forms. This is a boilerplate grid system that excels at being the best at and just that. The goal is to help designers focus on designing. The math is already taken care of.

High resolution

1140, 960, 1600? Choose your max resolution and breakpoints. Avoid unnecessary white space by customizing your margins. The grid isn't just responsive, you also have the option for a fluid layout.

Custom Column Size

Set the max-column size in your sass file, and it will generate a responsive grid system. So whether you want to work with a 16 grid layout or a 12 grid layout, zGrid gives you options.
